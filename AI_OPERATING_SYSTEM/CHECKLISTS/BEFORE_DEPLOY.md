## НАЗНАЧЕНИЕ

Проверка перед деплоем в production.

Не деплоить без проверок.  
Не деплоить без бэкапа.  
Не деплоить без плана отката.

## ПРОВЕРКИ

### 1. Бэкап
- [ ] Бэкап Supabase (pg_dump или dashboard)
- [ ] Бэкап n8n workflows
- [ ] Бэкап файлов (/var/www/nestandart/)
- [ ] Бэкап .env на VPS
- [ ] Метка времени сохранена

### 2. Код
- [ ] Все тесты проходят
- [ ] Линтер не ругается
- [ ] Сборка проходит без ошибок
- [ ] Docker образы собираются
- [ ] Миграции применены (если есть)
- [ ] Нет секретов в коде

### 3. Конфигурация
- [ ] .env на VPS актуальный
- [ ] nginx конфиг проверен
- [ ] Docker compose актуальный
- [ ] SSL сертификат действителен
- [ ] Порты открыты правильно (UFW)

### 4. Документация
- [ ] CHANGELOG.md обновлён
- [ ] Версия обновлена
- [ ] План отката готов
- [ ] Команда отката проверена

### 5. Мониторинг
- [ ] Health check работает
- [ ] Логи доступны
- [ ] Алерты настроены (если есть)
- [ ] UptimeRobot добавлен (если нужно)

## ПЛАН ОТКАТА

Перед деплоем обязательно иметь:

```bash
# Откат бэкенда
docker compose rollback kote-backend

# Откат фронтенда
git checkout HEAD~1 -- nestandart-phuket/
scp -r nestandart-phuket/ root@VPS:/var/www/nestandart/

# Откат БД
psql -U postgres -d nestandart < backup_YYYYMMDD.sql

# Откат n8n
docker exec kote-n8n n8n import:workflow --backup=backup_YYYYMMDD.json
```
