# MULTI-MARKET

## Принцип

Один бренд. Один код. Одна база данных. Много рынков.

Новый рынок — это строка в таблице `markets`, а не новый репозиторий.

---

## Как добавить новый рынок

### 1. База данных

```sql
insert into markets (id, name, active, tg_bot, wa_phone, domain)
values ('bali', 'Бали', true, 'nestandart_bali', '62XXXXXXXXX', 'nestandart-bali.ru');
```

### 2. Наполнить данными

Добавить туры, контент, базу знаний — всё с `market_id = 'bali'`.

### 3. Сайт

Добавить `bali` в `shared/markets.js`. Сайт подхватит автоматически.

### 4. HQ

Переключатель рынков в сайдбаре уже работает через `activeMarket`.

---

## Правила разработки

- Каждая таблица обязана иметь колонку `market_id`
- Никаких хардкодов названий рынков в логике — только через `market_id`
- RLS в Supabase фильтрует данные по рынку автоматически
