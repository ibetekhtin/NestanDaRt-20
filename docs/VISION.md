## 1. Миссия

Самый качественный сервис покупки классических экскурсий — сначала Пхукет, затем весь мир.
КотЭ — универсальный цифровой помощник путешественника: помогает экономить **деньги, время и нервы**.
Главный принцип: **сначала помочь человеку — потом продать.**

                                  │  САЙТ        │  nestandart.online

        Гость любым путём покупает 2+ экскурсии → доволен → отзыв с фото

Тактика продажи: пакет **«1 морской + 1 сухопутный + авторский тур»**.

## 5. Архитектура (проверено аудитом 2026-07-02)

```
Клиент ── Telegram ──► n8n «AI Агент с памятью» + рукава (Бронь, Память, дожим, напоминания, отзывы, reconcile)
   │                        │ service_role (env), RPC: get_kote_context / bot_upsert_client / …
   ├── САЙТ ───────────────►│ anon: только чтение витрины (tours/packages/knowledge)
   ├── PWA ── /api/leads ──►│
   │                   FastAPI backend (kote-backend :8000, X-Kote-Secret)
   │                        │ service_role
   └── БАЗА (админ) ───────► Supabase (cmmdrhususjuadqzyssc): 16 таблиц, RLS, SECURITY DEFINER RPC
                             ЮKassa ↔ /pay/create /pay/webhook /pay/reconcile
```

Безопасность (после lockdown-миграции): бизнес-RPC исполняет **только service_role**; у anon — read-only витрина; внутри RPC второй рубеж — sha256-гейт KOTE_SECRET; секреты только в `.env`/n8n credentials.

## 6. Этапы

1. **Сейчас — Пхукет как эталон.** Надёжность «как часы»: бронь → оплата → CRM без потерь. Вычистить долги (см. CODE_AUDIT).
2. **Треугольник дожат.** Ссылки между сайтом ↔ PWA ↔ ботом, сквозная аналитика, СБП в личном кабинете.
3. **Контент-комбинат.** Систематический трафик из соцсетей.
4. **Масштабирование рынков.** Возврат Паттайи/Вьетнама из архива, далее Бали/Дубай — только данными.
5. **Экосистема лояльности.** ЛК с балансом, СБП 1.5%, токен «НО».
